# How to install cae-cli

- To install cae-cli, you need Java 11 or higher installed on your machine.

## Step 1: Download the latest version of cae-cli

- Download the project from the link below:

<ButtonComponent href="https://github.com/clean-arch-enablers-project/cae-cli/archive/refs/heads/main.zip">
  Download
</ButtonComponent>

## Step 2.1: Install on Windows

- Extract the download zip to a folder
- Run the `cae-cli-installer.exe` file 
- Add %CAE_CLI_HOME%  to your system's or user's Path environment variable

</div>

<div id="windows" style={{

## Step 2.2: Install on macOS

- Extract the download zip to a folder
- Run the `cae-cli-macos-installer.sh` in sudo mode
- Add theses exports on `.zshrc` or `.bash_profile` file:
```bash
export CAE_CLI_HOME="$HOME/cae"
export CAE_META_STRUCTURE_TEMPLATES_PATH="$HOME/cae/file-templates"
```
- Restart terminal with `source ~/.zshrc` or `source ~/.bash_profile`

</div>

## Step 3: Verify the installation

- Run the command `cae-cli ls` on your terminal
- If everything is ok, you will see something like this:
